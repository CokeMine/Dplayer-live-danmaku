# Dplayer-live-danmaku

A Easy danmaku backend for Dplayer Live Mode using websocket.

通过websocket简单实现Dplayer直播的弹幕后端

所需环境：Python 3.7+ , Module: websockets

```
apt install python3 python3-pip
pip install websockets
```
